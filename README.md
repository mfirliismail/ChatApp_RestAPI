# ChatApp_RestAPI

## instalation
1. clone a repo,
2. install all dependencies with `npm install`
3. connect to database local in config.json
4. set PWD_TOKEN in .env file like `PWD_TOKEN=(the keyword)`
4. run with `npm start`
